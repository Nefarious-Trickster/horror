# Neon Ruins - Platformer Prototype

A fast 2D platformer built in plain HTML5 Canvas.

## Play
```bash
python -m http.server 8000
```
Open `http://localhost:8000/web/`.

## Controls
- Move: `A/D` or `←/→`
- Jump: `W`, `↑`, or `Space`
- Dash: `Shift`

## Mechanics
- Coyote time + jump buffer for responsive controls.
- Double jump with animated air burst.
- Directional dash with cooldown trail.
- Wall slide + wall jump.
- Moving platforms and collectible shards.
- Patrol drones and hazard pits.
